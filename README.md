# Kleidi

https://kleidi.org/

## Setup

### Pré-requis

- Ruby 2.6.6

### Installation

- Cloner le repo
- Installer les dépendances avec `bundle install`
- Lancer avec `middleman` puis aller sur http://localhost:1313

### Déploiement

- Compiler le site avec `middleman build`
- Copier le contenu du dossier `build` sur le serveur FTP.
